## 🛠️ 주요 기능
  1. 항공권 조회 및 결제
  2. SNS 커뮤니티
  3. 이미지검색을 통한 여행지 추천
  4. API 사용 페이지
  5. 로그인 시큐리티 및 소셜로그인
  6. JWT 토큰 사용 보안처리
